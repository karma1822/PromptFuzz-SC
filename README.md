PromptFuzz-SC
PromptFuzz-SC 是一个用于测试和评估大语言模型（LLM）安全性的研究框架。通过义空间和字符空间的双层变异策略，结合智能搜索算法，自动发现和评估 LLM 的越狱漏洞。

PromptFuzz-SC is a research framework for testing and evaluating the security of Large Language Models (LLMs). Leveraging a dual-space mutation strategy across semantic and character spaces, integrated with intelligent search algorithms, it enables automated discovery and assessment of jailbreak vulnerabilities within LLM systems.
